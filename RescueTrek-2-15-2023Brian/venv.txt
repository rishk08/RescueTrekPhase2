Notes on how to create a virtual environment for Python

Install venv:
pip3 install virtualenv

create venv folder:
virtualenv venv -p python3 

activate virtual environment (Git bash terminal for Windows):
source venv/scripts/activate (Windows)
OR 
source venv/bin/activate (Linux/MacOS)

This command will list all installed packages into requirements.txt for easy replication:
pip3 freeze > requirements.txt
